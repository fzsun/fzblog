为什么 integer programming 很难解决？

IP 问题是 NP-hard.

简单的概率组合题，举一例：A 和 B 轮流掷硬币，每轮同时掷两枚，谁先掷到双 head 谁赢。假设 A 先开始。求 A 赢的概率。

知道 geometric distribution 可以直接给答案。我又是现推，思路是先找到每轮 A 赢的概率，再求和，是个几何级数，求个极限。当时又忘了几何级数的求和公式（记性太差），准备硬推，被面试的拦住。

经典的 uncacitated facility location problem，求 formulation。

无数 IP 教材的例子题。就不赘述了。我的限制 x，y 关系的约束是 aggregated 版本。在此基础上有一家公司还问了，假如把这个约束替换为 decomposed 版本，怎么评价这两个 formulation 的好坏。

解 IP 有哪些方法？

方法一大堆就不罗列了。小插曲是当我说 lagrangean relaxation 时，面试的顺便问了这个怎么 work 的，顺着思路又问这个方法下怎么判定 complicated constraint 和 nice constraint。

newsvendor problem 的最优解是什么？假设有 salvage cost，没有 holding cost，给定了需求的 cumulative distribution function。

当时不太熟悉这个模型，硬建立了一个 stochastic IP 模型。当面试的说其实这个有解析解，我有点小崩溃，推不出来只能跳过。解析解可以参考这篇文章。

问了一个 transportation 问题的建模和解法，实质是变体 bin packing 问题。

恰好知道一个很高效的启发式算法，我说，要不要先给你解释下再说建模的事。面试的同意，就大概讲了 first fit decreasing 的思路，顺便告诉他印象中这个算法的 worst case bound 在 10% 以内，非常好了。他问，为什么这个启发式算法有时达到不了最优，举例说明。之后就是怎么建这个问题的数学模型，顺带又问了个比较两种 formulation 好坏的问题。

fit 曲线部分：继而问了使用什么模型（线性回归），线性回归有哪些假设（我答了线性，error term 是 mean 0 并且 iid），如何解释这个模型，如果价格变量的系数是正表示什么意思。问如何在模型里加入季节性。我说加入12个 binary variable 即可，他问是否可以减成11个 variable，我说可以。然后让我解释这11个变量的每个变量的系数的含义。问还有哪些变量可以加入以更好的预测需求。

  1. 一个六面骰子，点数从1-6，每个面概率相等。你有三次掷骰子的机会，每掷一次可以选择停止并接受本轮点数为奖励或掷下一次。如何制定最优策略使得奖励的期望最大化。
    我的解题思路：动态规划。递归式是

V(t)=∑i=1616max⁡{i,V(t−1)}V(t)=\sum_{i=1}^6 \frac{1}{6} \max \{i,V(t-1)\} V(t)=i=1∑6​61​max{i,V(t−1)}

$V(1)=3.5$
$V(2)=(3.5 \times 3+4+5+6)/{6}=4.25$
$V(3)=(4.25 \times 4+5+6)/6=4.67$
因此三次机会的最佳策略是。第一次机会，投到5或6停止，否则投第二次。第二次机会，投到4或5或6停止，否则投第三次。第三次是最后一次，没有策略。如此得到的最大化点数期望为4.67。

    找 tight formulation，strong valid inequality 等。

  2. 一维数据 k-means 的算法。我提了几个：

    1. 用0-1变量表示每个数据点是否在某个cluster里，然后用数学规划 formulate 出 model。
    2. 一维数据的聚类实际是找数轴上的分界点。那么可以用 neighborhood search 方法，初始时平分数轴，每次左右移动分界点都要保证有所改进，直至不能改进。
    3. 动态规划：如果有 T 个 cluster，n 个数据点，令 $f(x,n)$ 为剩余 n 个点时，聚合左边 x 个点的成本（距离中心点距离之和），那么 $V_t(n)=\min_x \{f(x,n)+V_{t+1}(n-x)\}$。 复杂度为 $O(n^2 T)$。

  4. 排班问题，如何实现不能出现休息一天，工作一天，休息一天，即不能出现010的情况。$x_1+x_3 \geq x_2$
