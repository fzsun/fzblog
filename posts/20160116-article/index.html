2015年观影十佳
Jan 16 2016

2015年实看了116部片，算是观影小爆发的一年吧。观影口味似乎回归了温情系，就是那种扎实沉稳叙事的，带一点亲情爱情的关怀的片子。不过 cult 片也一直都是我的菜。前几天跟刚上大学的表妹聊起电影，问到我喜欢什么类型的电影。她说一定是高智商烧脑片吧。回想到，初进大学，那会儿人人上常有分享十大高智商烧脑电影的，大家都很乐此不疲。觉得这才叫电影，就是跟爆米花片不一样。这类片子确实挺神的，影片严丝合缝的细节，以及隐藏线索也是大家津津乐道的谈资。也喜欢这类片，但不会到爱不释手的地步。觉得更好的片子，往往并不烧脑，它们要么有诗意，要么有情怀，或者仅仅是把一个类型做到极致也是很好的。从15年看过的100多部片子里挑出了10部佳作，都在某些地方有值得称道的地方，也是我愿意反复观看的片子。于是也就有了这篇日志。

在那之前，谈谈跟我的观影有关的一些有趣的数字吧。暑假重新整理了一道豆瓣上的观影标签，便于以后查找。花了一天给每个看过的电影补足四个标签：制片国家，导演，观影年月，影片的年份。于是也浮现出一些有趣的数字。

2012年至2015年月观影数量
image

每月的观影数量显现出很明显的 pattern。大致上，每年的1，7，12月都是观影的高峰期，显然对应的是寒暑假的空闲时间。学期中间则比较忙碌，每周不到一部电影。总体上看的电影越来越多了，15年之前平均每年60部，15年则翻了一番。豆瓣上全部标记的已看电影也达到了465部。

虽然对外宣称自己很喜欢看电影，其实花在电影上的时间不比很多人。知道自己的时间紧张，所以几乎不看剧集。然而在过去的一年，也追完了一部美剧《冰血暴2》。原因自然是冲着最喜欢的科恩兄弟。题外话是，第二季没有第一季精彩。这几年几乎看完了科恩兄弟的所有作品，20来部，也并非每部都是上乘之作，不乏平庸作品。也许看片越多，越是带有批判性地去看片，算是成长了。

15年尝试去补全的第二个导演是库布里克，已经看了他的七部片。据说老库在世长片一共12部，已看了一大半了。看过的七部片，着实每部都是影史经典。《光荣之路》《闪灵》《奇爱博士》《2001太空漫游》《巴里林登》《全金属外壳》《发条橙》，题材迥异，但每一部都值得津津乐道一番。老库炉火纯青。16年应该会至少把老库的《大开眼戒》《洛丽塔》《斯巴达克斯》看了。

观影量按导演统计
image

卢米埃尔这么多片主要还是因为数据太好刷，在电影发明的年代，时长都不过几分钟。其他几个导演的观影量都是情理之中，差不多就是这些年来自己总体的观影口味。

接下来希望可以补全的导演有：斯科塞斯，希区柯克，李安，波兰斯基，伍迪艾伦。

观影量按影片年份统计
image

第三个统计则是关于所看的片子的年份。越近的片子看得越多，对那些老片实在是有些抗拒的。估计这个趋势长期都不会改变。比较有好感的年份大抵从90年以后始。希望可以逐渐往前扩展到80年代，70年代。

最后一个统计量是关于国别。

观影量按国别统计
image

意大利太寒酸。有种感觉，现在美国的文化影响力比几十年前更加强盛了。几十年前也许还是百家争鸣，现在全世界都在看好莱坞。可能世界也只能在艺术电影上有些立足地。

接下来是我的2015年度十佳。

1.光猪六壮士 （1997 彼得·卡坦纽）

image

这部97年的英国喜剧片子当时也是红极一时，叫好亦叫座。提名了98年奥斯卡最佳影片，最佳导演，最佳原创剧本。片子大体描述了英国工业衰退的场景，悲凉，但工人们也苦中作乐。叙事很流畅，很多黑色幽默贯穿其中。我觉得最有趣的设定是，一群男的非常认真地排练脱衣舞给女的看。起先这么做是生活所迫，但逐渐的，大家这么做更像是为了完成理想，为了兄弟的情谊。荒诞可笑的背后，竟也被感动到了。

2.光荣之路 （1957 斯坦利·库布里克）

image

老库的这部片子着实是讽刺的一座高峰，一部反战的战争片。看完好无奈，大概错的是这个世界。精巧的剧情、讽刺和出人意料，太喜欢了。观众以为军事法庭上上校的动情辩护可以拯救士兵，然而还是被判有罪；观众以为机智上校的说动了将军，满心期待刑场最后一刻冒出“刀下留人”，结果士兵被果断处决，残忍地不留一点念想。德国女孩的歌唱，把一部战争片硬生生变成了催泪片。

3.饮食男女 （1994 李安）

image

李安的《饮食男女》应该是很多人的至爱。把中国的老传统描写的如此细腻，就是一个纯粹的文化人。美食与爱，都不可辜负。

4.土拨鼠之日 （1993 哈罗德·雷米斯）

image

这部片是好片，就是海报怎么看怎么二，就像洗剪吹小哥在凹造型。推荐它的原因是，构思实在太棒了。从一个很简单的设定引发了无限脑洞，说明了一些道理，不做作。

5.背靠背，脸对脸 （1994 黄建新 / 杨亚洲）

image

这部豆瓣评分高达9.0的片子只有寥寥不到5000人打了分，算是非常小众的高分片。看完以后，在我心目中就是跟《霸王别姬》《鬼子来了》一样高度的片子了。值得一看。海报很乡土，大概因此让很多人错过了它。中国官场的一个缩影，揭露到底子了。很犀利。

6.奇爱博士 （1964 斯坦利·库布里克）

image

第二部推荐的老库片。也是通篇的讽刺，慢条斯理的把疯癫的人性表现出来，到处充斥着黑色幽默。几乎就是完美了。

7.老炮儿 （2015 管虎）

image

这部是在国内院线看的，很中二，但我竟然被中二感动得热泪盈眶，真的哭了。江湖不是原来那个江湖了。喜欢父子言和戏，鸵鸟出笼戏，六爷单刀赴会戏。 我还是把本应该会给四星的片子放在这里了，就是因为爱不释手，这片有瑕疵，还没到完美。

8.狩猎 （2012 托马斯·温特伯格）

image

关于人性黑暗的寓言。男主演技要爆棚。最喜欢教堂平安夜弥撒那段，男主回首的双眼，愤怒无奈无助。谣言下，信任太容易崩塌。不应该有任何倾向，弱者也不一定天然就是对的。这个片子启示很大。

9.哪啊哪啊神去村 （2014 矢口史靖）

image

比较轻松休闲的片子吧，简单质朴，有笑有泪，总体观感类似去年推荐的横道。

10.王牌特工：特工学院 （2014 马修·沃恩）

image

在院线看的，当时就惊为天人，有范儿，很嗨。相当cult，看得爽翻。教堂乱战戏，后来反复看了好多遍，喜欢死了，辅以Lynyrd Skynyrd的Free Bird，那段solo燃爆了。最后全世界爆礼花，交响乐响起，心里激动的都要跳起来了。

关于为什么我的十佳不是新片

一直觉得自己将长期处于观影初级阶段。现阶段的主要任务当然是如饥似渴地涉猎各个类型题材的好片。相对于仅仅一年的新片，显然老片更容易淘出好看的片子来。

一个有意思的事是，这几年的豆瓣观影平均分基本在7.7附近，每年上下浮动不超过0.2。我自己的打分原则，除非爱不释手，各方面都很好的，觉得以后会回看的，才会打5星。给电影的基准分是3星，如是佳作，就加一星。两星很少打，现阶段无暇看烂片。一星就更少了。如此看来，这几年看了很多佳作。估摸着，等阅片量上了1000，平均打分下了7，对电影的理解又是另一番境界了吧。

十佳很少放新片的另一个原因，能挑出十佳新片起码得看50+的新片，这个现阶段做不到。没那个时间，research 很忙的。所以现在特羡慕上班族（特指朝九晚五不加班的），至少每天晚上可以看一部了吧，一年就是300+部，好过瘾！
